# dream-gripper
The Dream Gripper is a hardware project at Dream Machines.

The main scope when designing this gripper, was to pick up anything from the size of an o-ring, to a coffee mug. At the same time, the gripper was intended to be easily reproduceable and reliant only on relatively standard materials. Also, the gripper was designed with a linear rail as a platform in use, results with a pivot style setup were not tested.
